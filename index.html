Bengala com Sensor de Obstáculos

Este projeto consiste no desenvolvimento de uma bengala inteligente equipada com tecnologia assistiva para aumentar a autonomia de pessoas com deficiência visual. O objetivo principal é detectar obstáculos aéreos ou terrestres que a bengala convencional não alcança, prevenindo colisões e quedas. Para isso, utilizamos a plataforma Arduino integrada a um sensor ultrassônico, que mede a distância de objetos em tempo real, e um sistema de alerta sonoro (buzzer) ou vibratório. O resultado esperado é um dispositivo de baixo custo e alta eficiência que proporcione mais segurança, confiança e mobilidade ao usuário em seus deslocamentos diários.

Explicação do Projeto

O Problema: Desafios da Mobilidade

Pessoas com deficiência visual enfrentam barreiras diárias que vão além da falta de sinalização adequada. A bengala guia tradicional, embora essencial, possui uma limitação física: ela só detecta obstáculos que estejam em contato direto com o solo. Isso deixa o usuário vulnerável a obstáculos aéreos, como galhos de árvores, placas e outros objetos suspensos, que podem causar acidentes na região do tronco e do rosto. Além disso, a incerteza sobre o ambiente gera insegurança emocional e limita a liberdade de locomoção.

A Importância da Tecnologia Assistiva

A tecnologia assistiva atua como uma ponte entre a limitação e a autonomia. No contexto da deficiência visual, ferramentas inteligentes não substituem os sentidos humanos, mas os ampliam. Ao integrar sensores e sistemas eletrônicos em objetos comuns, transformamos um instrumento simples em um dispositivo interativo, capaz de interpretar o ambiente e fornecer informações em tempo real, aumentando o tempo de reação do usuário e sua percepção espacial.

Justificativa

A justificativa deste projeto está na necessidade de promover inclusão social e acessibilidade de forma democrática. Desenvolver uma bengala inteligente utilizando componentes de baixo custo demonstra que é possível criar soluções eficientes sem depender de tecnologias inacessíveis.

  • Segurança: Reduzir o risco de colisões com obstáculos.
  • Autonomia: Permitir maior independência na locomoção.
  • Dignidade: Garantir o direito de ir e vir com mais confiança.

Funcionamento do Sistema

O sistema funciona da seguinte forma:

•O sensor ultrassônico emite ondas sonoras

•Essas ondas atingem um objeto e retornam

•O Arduino calcula a distância com base no tempo de retorno

•Se o objeto estiver próximo, o sistema ativa o alerta

•O alerta pode ser sonoro (buzzer) ou vibratório

Código Utilizado

// --- Definição dos Pinos ---

const int TRIG = 12; // Pino de envio do ultrassom (Gatilho)

const int ECHO = 11; // Pino de retorno do eco

const int BUZZER = 10; // Pino do alarme sonoro

void setup() {

// Configuração inicial (roda uma vez ao ligar)

pinMode(TRIG, OUTPUT); // O pino TRIG envia sinal (saída)

pinMode(ECHO, INPUT); // O pino ECHO recebe sinal (entrada)

pinMode(BUZZER, OUTPUT);// O pino BUZZER emite som (saída)

// Inicia comunicação serial para testes (opcional)

Serial.begin(9600);

}

void loop() {

// --- 1. Limpeza e Disparo do Pulso ---

digitalWrite(TRIG, LOW);

delayMicroseconds(2);

// Envia um pulso de 10 microsegundos

digitalWrite(TRIG, HIGH);

delayMicroseconds(10);

digitalWrite(TRIG, LOW);

// --- 2. Leitura do Eco ---

// A função pulseIn espera o pino ir para HIGH e conta o tempo

long duracao = pulseIn(ECHO, HIGH);

// --- 3. Cálculo da Distância ---

// Distância = (Tempo * Velocidade do Som) / 2

// Velocidade do som = 0.034 cm/us

int distancia = duracao * 0.034 / 2;

// --- 4. Lógica de Alerta ---

// Só atua se detectar algo entre 0 e 100cm (1 metro)

if (distancia > 0 && distancia <= 100) {

if (distancia < 20) {

// PERIGO IMINENTE (< 20cm): Som contínuo

digitalWrite(BUZZER, HIGH);

} else {

// ALERTA DE PROXIMIDADE: Bipes intermitentes

// Quanto menor a distância, mais rápido bipa

digitalWrite(BUZZER, HIGH);

delay(distancia * 5); // O tempo do bipe depende da distância

digitalWrite(BUZZER, LOW);

delay(distancia * 5); // O intervalo também

}

} else {

// Se estiver longe (> 100cm), silêncio total

digitalWrite(BUZZER, LOW);

}

// Pequena pausa para estabilidade

delay(50);

}

Resultados Obtidos

O dispositivo apresentou funcionamento satisfatório durante os testes. Foi possível detectar obstáculos a uma distância média entre 50 cm e 100 cm. O buzzer foi acionado corretamente sempre que um objeto se aproximava da distância definida. Foram observadas pequenas variações dependendo do tipo de superfície e do ambiente.
