Bengala com Sensor de Obstáculos

Projeto para ajudar pessoas com deficiência visual.

Introdução

Este projeto consiste no desenvolvimento de uma bengala inteligente equipada com tecnologia assistiva para aumentar a autonomia de pessoas com deficiência visual. O objetivo principal é detectar obstáculos aéreos ou terrestres que a bengala convencional não alcança, prevenindo colisões e quedas.

Para isso utilizamos Arduino, sensor ultrassônico e um sistema de alerta sonoro ou vibratório. O resultado esperado é proporcionar mais segurança e mobilidade ao usuário.

1. O Problema: Desafios da Mobilidade

Pessoas com deficiência visual enfrentam diversas dificuldades para se locomover. A bengala tradicional detecta apenas obstáculos que estão em contato com o solo.

Isso deixa o usuário vulnerável a galhos de árvores, placas e objetos suspensos.

2. A Importância da Tecnologia Assistiva

A tecnologia assistiva ajuda pessoas com deficiência a terem mais autonomia.

Com sensores e componentes eletrônicos é possível melhorar a percepção do ambiente.

7. Funcionamento do Sistema

O sensor ultrassônico envia ondas sonoras.

Quando as ondas encontram um objeto elas retornam ao sensor.

O Arduino calcula a distância e ativa o alerta quando necessário.

8. Código Utilizado

// ===== HC-SR04 =====
const int trigPin = 9;
const int echoPin = 10;


// ===== LED RGB (cátodo comum) =====
const int ledR = 3;
const int ledG = 5;
const int ledB = 6;


// ===== Motor =====
const int motorPin = 11;


// ===== Buzzer =====
const int buzzerPin = 12;


long duracao;
float distancia;


void setup() {
  Serial.begin(9600);


  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);


  pinMode(ledR, OUTPUT);
  pinMode(ledG, OUTPUT);
  pinMode(ledB, OUTPUT);


  pinMode(motorPin, OUTPUT);
  pinMode(buzzerPin, OUTPUT);
}


void loop() {


  // Leitura do HC-SR04
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);


  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);


  digitalWrite(trigPin, LOW);


  duracao = pulseIn(echoPin, HIGH, 35000);


  distancia = duracao * 0.0343 / 2.0;


  Serial.print("Distancia: ");
  Serial.print(distancia);
  Serial.println(" cm");


  // ===== LONGA DISTÂNCIA (250 a 400 cm) =====
  if (distancia > 250 || distancia <= 0) {


    // Verde
    digitalWrite(ledR, LOW);
    digitalWrite(ledG, HIGH);
    digitalWrite(ledB, LOW);


    digitalWrite(motorPin, LOW);
    noTone(buzzerPin);


    delay(100);
  }


  // ===== MÉDIA DISTÂNCIA (100 a 250 cm) =====
  else if (distancia > 100) {


    // Amarelo
    digitalWrite(ledR, HIGH);
    digitalWrite(ledG, HIGH);
    digitalWrite(ledB, LOW);


    // Vibração leve
    digitalWrite(motorPin, HIGH);
    tone(buzzerPin, 1000);
    delay(120);


    digitalWrite(motorPin, LOW);
    noTone(buzzerPin);
    delay(500);
  }


  // ===== CURTA DISTÂNCIA (30 a 100 cm) =====
  else if (distancia > 30) {


    // Vermelho
    digitalWrite(ledR, HIGH);
    digitalWrite(ledG, LOW);
    digitalWrite(ledB, LOW);


    // Vibração média
    digitalWrite(motorPin, HIGH);
    tone(buzzerPin, 2000);
    delay(80);


    digitalWrite(motorPin, LOW);
    noTone(buzzerPin);
    delay(150);
  }


  // ===== MUITO PERTO (< 30 cm) =====
  else {


    // Vermelho
    digitalWrite(ledR, HIGH);
    digitalWrite(ledG, LOW);
    digitalWrite(ledB, LOW);


    // Vibração forte
    digitalWrite(motorPin, HIGH);
    tone(buzzerPin, 3500);


    delay(30);
  }
}

10. Análise e Discussão

Pontos positivos:

  • Baixo custo
  • Fácil montagem

Limitações:

Possíveis melhorias:

  • Mais sensores
  • Alerta vibratório
